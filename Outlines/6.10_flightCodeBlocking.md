## Code Components
High level blocks of code most likely organized into specific files 
- Listener: Functions that will constantly run on independent threads that will listen for desired RS-422 signals, and return a value when those signals are found 
    - Connections: IN-Craft, OUT-Ctrl Logic
    - Filetype: Python script/class
- Ctrl Logic: Main loop of the software, waits for values from listener and uses decision trees to execture pre-pregramed series of instructions based on the signals and time since signal receipt 
    - Connections: IN-Listener, OUT-Relay Manager, PXIe Ctrl
    - Filetype: Python script/class
- Relay Manager: Maintains a open serial comm port to relay board, when open or close function is called sends the neccesary signal to that relay
    - Connections: IN-Ctrl Logic, OUT-Relay Board
    - Filetype: Python script/class
- PXIe Ctrl: Opens IP communication to the PCIe cabinet & sends the signals to the function generators using the pickering drivers with the desired parameters for the phased array 
    - Connections: IN-Ctrl Logic, Phased Array OUT-PXIE Cabinet
    - Filetype: C-script using Pickering drivers
- Phased Array: Stored configurations of phased arrays that include the frequency, voltage amplitude, and phase offset between pairs
    - Connections: OUT-PXIe Ctrl 
    - Filetype: .csv of stored array configurations
