The pickeringInterface.py handles all interfacing to update the waveforms generated by the function generators. It consists of two functions the initPXIE() and updateWaveform(). 
NOTE: You MUST install the Pickering Direct IO drivers to your machine they are located in this repo under \PickeringManuals\Pickering_PXI_Driver_Package. This set of drivers handles the conversion from the C wrapper that is the python package to the IO interfacing. 

initPXIE() - Run this function in the main loop
    RETURNS - tuple consisting of two arrays
